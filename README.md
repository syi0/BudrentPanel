# BudrentPanel - jak odpalić projekt na komputerze

Wymagania - pobrany [Node.js](https://nodejs.org/en)

Aby odpalić program, należy uruchomić 2 konsole w głównej ścieżce projektu (BudrenPanel)

Najpierw odpalamy backend (komenda po komendzie):

```
cd backend
npm i
node server.js
```

Następnie w kolejnej konsoli uruchamiamy frontend:

```
cd frontend
npm i
npm run dev
```

Aby wejść w aplikacje, należy wejść na strone localhost:5173
